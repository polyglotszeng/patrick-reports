📚 10 个推测模块（基于趋势与同类课程）

MODULE 0
课程介绍与心智模型
  • AI-Infused SaaS vs 传统 SaaS
  • Claude Code vs Cursor / Bolt / Lovable
  • 最终交付物：1 个完整 SaaS 应用
MODULE 1
项目规划与架构
  • Idea 选择：市场需求 → PRD
  • Tech stack (Next.js + Supabase + Stripe + shadcn-ui)
  • 数据模型 + Pricing model
MODULE 2
Claude Code 基础
  • 安装与配置 (CLI / VSCode / Cursor)
  • CLAUDE.md 与 memory 文件
  • Slash commands + subagents
  • MCP (Model Context Protocol) 服务
MODULE 3
项目脚手架与基础架构
  • Claude Code 生成 Next.js 14 app router
  • 配置 shadcn-ui + Tailwind 主题
  • Supabase (auth + database + storage)
  • Stripe (subscriptions + one-time)
MODULE 4
核心功能实现
  • 用户认证 (sign up / login / OAuth / magic link)
  • Dashboard (基于角色的 UI)
  • CRUD (产品/订单/订阅)
  • AI 集成 (OpenAI / Anthropic API + streaming)
MODULE 5
AI 功能深度集成 ⭐
  • 嵌入 Claude API 作为产品功能
  • Prompt engineering + streaming UI
  • 成本控制 (caching / rate limiting)
  • Guardrails (防 prompt injection)
MODULE 6
UI/UX 打磨
  • Design system + components library
  • 动效与微交互 (framer-motion)
  • 暗色主题 / 响应式 / 无障碍
  • Landing + pricing page 设计
MODULE 7
部署与生产化
  • Vercel 部署 + preview environments
  • 环境变量管理
  • Monitoring (Sentry / PostHog)
  • 性能优化 (Core Web Vitals)
MODULE 8
营销与增长
  • SEO (Next.js metadata + sitemap)
  • 启动策略 (PH / Twitter / Reddit)
  • 分析 (Mixpanel / Amplitude)
  • A/B 测试基础
MODULE 9
商业化与扩展
  • 支付墙设计 (trial / paywall / upgrade)
  • Customer success + onboarding
  • 反馈循环与迭代
  • AI agent 化（让产品自己做生意）
