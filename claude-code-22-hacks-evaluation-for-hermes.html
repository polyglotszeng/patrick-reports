🚀 三个新增 / 立即可做的机会

1. 语音输入做上下文注入 (Hack #3)

视频原意: 比打字快, 注入 context 时用 voice

Patrick 现况: Think & Quest 已加 voice input/output (Web Speech API)

新机会: 给 Hermes 加 voice-to-text-skill — 通过 Web Speech API 或 macOS Dictation 把语音转成 prompt

实施: 1-2 天可做

2. 每次操作前 commit (Hack #5)

视频原意: 立刻 commit, 写 commit message, 失败可回滚

Patrick 现况: 已有 ~/.openclaw/scripts/git-auto-sync.sh

新机会: 把 git-auto-sync 改成主动模式 — Hermes 在执行高危操作前自动 commit (而不是事后) + 把 commit message 写详细 (含任务摘要)

实施: 改 1 个脚本 + skill_manage patch requesting-code-review skill

3. Skill-to-skill API (Hack #17)

视频原意: Skills 互相引用 + 共享数据, 避免重复 prompt

Patrick 现况: skill_view 看 SKILL.md + linked_files, 但 skill A 调用 skill B 不直接

新机会: 在 Hermes 内部加 skill_run("name", args) API — 让一个 skill 显式调用另一个 skill

实施: 中等难度, 需 Hermes 内部支持
