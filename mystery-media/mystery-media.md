# 推理影视 10 部

## 1. Sherlock (BBC, 2010-2017) — Paul McGuigan 等
**核心理念**：现代伦敦 + 福尔摩斯 + Benedict Cumberbatch。4 季 + 特别篇。
- **Benedict Cumberbatch + Martin Freeman**。每集 90 分钟 = 电影级制作
- **创新**：iPhone + 短信 + GPS 替代电报；视觉化推理 = 屏幕文字 + 思维宫殿
- **范式贡献**：古典 IP 现代化方法论 / 90 分钟单集电影级

## 2. Murder on the Orient Express (2017) — Kenneth Branagh
**核心理念**：Kenneth Branagh 自导自演波洛。
- **Branagh + Penélope Cruz + Willem Dafoe + Judi Dench 等 12 巨星**。3.53 亿票房
- **范式贡献**：古典 IP 现代商业化 / 群像明星策略

## 3. Knives Out (2019) — Rian Johnson
**核心理念**：Daniel Craig 饰演 Benoit Blanc。致敬阿加莎 + 颠覆。
- **Daniel Craig + Chris Evans + Jamie Lee Curtis** 等
- **续集**：Glass Onion (2022) + Wake Up Dead Man (2025)
- **范式贡献**：新本格 IP / 政治隐喻

## 4. The Girl with the Dragon Tattoo (2011) — David Fincher
**核心理念**：Fincher 翻拍瑞典版。
- **Daniel Craig + Rooney Mara** 获奥斯卡女主提名
- **Trent Reznor + Atticus Ross 配乐获奖**
- **范式贡献**：Nordic Noir 美国化

## 5. True Detective S1 (2014) — Cary Fukunaga
**核心理念**：McConaughey + Harrelson。路易斯安那 + 邪教 + 17 年。
- **McConaughey 凭此获奥斯卡**。6 集单季 = "限定剧" 范式
- **6 分钟一镜到底** (S1E4) 名场面
- **范式贡献**：限定剧 (anthology) / 哲学对话剧本

## 6. Zodiac (2007) — David Fincher
**核心理念**：真实案件 + 执着 vs 放弃。1969-1983 黄道带杀手。
- **Jake Gyllenhaal + Robert Downey Jr. + Mark Ruffalo**
- **范式贡献**：真实罪案电影 / 执念主题

## 7. The Maltese Falcon (1941) — John Huston
**核心理念**：黑色电影经典。Humphrey Bogart 第一主角。
- **Bogart + Mary Astor**。film noir 范式确立
- **范式贡献**：film noir 视觉范式

## 8. The Long Goodbye (1973) — Robert Altman
**核心理念**：Altman 改编钱德勒。Elliott Gould 饰 Marlowe。
- **70 年代反英雄 Marlowe** + 蒸汽朋克重塑
- **范式贡献**：古典硬汉现代化

## 9. Sherlock Holmes (2009-2011) — Guy Ritchie
**核心理念**：RDJ + Jude Law。动作 + 福尔摩斯 + 蒸汽朋克。
- **2 部票房 5 亿+**
- **范式贡献**：古典 IP 动作化

## 10. Bad Kids / 隐秘的角落 (2020) — 辛爽
**核心理念**：紫金陈《坏小孩》改编。秦昊 + 王景春 + 3 童星。
- **豆瓣 9.0+ / 爱奇艺迷雾剧场**。3 个孩子策划犯罪
- **范式贡献**：中国推理剧巅峰 / 迷雾剧场范式
