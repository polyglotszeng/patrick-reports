🔗 跨界大师 (6 位覆盖 2+ 类型)

奇幻 + SF + 拉美
Ursula K. Le Guin — 地海 (奇幻) + 黑暗的左手 (SF) + 一无所有 (SF)。阴翳道家 = 通杀两界
SF + 推理
刘慈欣 + 紫金陈 — 都是中国当代头部。不同：宏大尺度 vs 城市场景。同：算法 / 系统 / 人性拷问
奇幻 + 推理
阿来 + 马伯庸 — 中国当代。都是"历史 + 文学 + 哲学"。马伯庸 = 把史料当推理素材
恐怖 + SF
H.P. Lovecraft + Mary Shelley — Lovecraft 宇宙恐怖 = SF 终点; Shelley 弗兰肯斯坦 = SF+恐怖双源
非虚构 + SF
Carl Sagan — 科普文学化 + 哲学化; Yuval Harari 全球史 + 后人类议题
拉美 + 推理
Borges — 拉美文学最高神 + 概念小说 (侦探 + 哲学 + 迷宫)。Cortázar 跳房子 = 拉美 + 实验叙事
