🔗 跨类型作家交叉 (6 位跨界大师)

类型 1: 奇幻 + SF
Ursula K. Le Guin — 地海 (奇幻) + 黑暗的左手 (SF) + 一无所有 (SF)。阴翳道家 = 通杀两界
类型 2: SF + 推理
刘慈欣 + 紫金陈 — 都是中国当代头部。不同：宏大尺度 vs 城市场景。同：算法 / 系统 / 人性拷问
类型 3: 奇幻 + 推理
阿来 + 马伯庸 — 中国当代。都是"历史 + 文学 + 哲学"。马伯庸 = 把史料当推理素材
类型 4: 文学 SF
Borges — "作家版 30" 中唯一纯文学 SF (短篇迷宫 / 巴别图书馆)。同时是博尔赫斯 = 拉美文学最高神
类型 5: 哲学 SF
Lem + Ted Chiang — 他者不可知性。Lem 长篇 / Chiang 短篇。同：1 个假设撑起整本书
类型 6: 系统奇幻
Sanderson + Rowling — 魔法系统工程师。Sanderson = 物理学化魔法 / Rowling = 学校 + 商业化
