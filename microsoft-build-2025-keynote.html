Microsoft IQ — 三层统一架构

为 Agent 提供实时、可靠、可溯源的推理基础，三层 IQ 统一协同：

🌐 Web IQ
外部信息层

持续索引网页、新闻、图片、视频，实时获取外部信息，quality / speed / cost 三个维度均为最佳

🔗 Fabric IQ
实时运营数据

组织文档、运营数据、数字孪生模型，耦合实时遥测（分钟级真实状态）

📋 Work IQ
规程文档

连接 SharePoint 中的 SOP 文档，从团队日常维护的真实来源获取，非静态快照

演示场景（电力公司应急）：① Web IQ 查 SF 电价 → ② Fabric IQ 查变电站实时状态 → ③ Work IQ 调应急 SOP → ④ Agent 完成任务
