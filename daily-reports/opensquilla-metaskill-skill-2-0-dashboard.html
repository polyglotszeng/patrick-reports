🔥 核心论点

🏗️ 微内核 + 插件架构

核心引擎（最小集）

  • 编排
  • 调度
  • 状态管理
  • 不做其他事

其余能力（全部插件）

  • 以插件形式运行
  • 在用户态独立
  • 可独立扩展 / 替换
"核心引擎只做最少的事，只进行编排调度和状态管理，其余的所有的能力全部是以插件的形式在用户态运行"

优势（操作系统类比）

  • 核心稳定
  • 插件灵活
  • 易扩展
  • 单点失败不传播

MetaSkill 源文件结构

MetaSkill 文件:
  - 元数据 (name / description / use case)
  - 步骤 1: 追问题（缺什么信息）
  - 步骤 2: 可行性判断
  - 步骤 3: 拆子任务
  - 步骤 4-N: 串接子 Skill
  - 输出模板 (Markdown 格式)
  - 安全提醒

💬 关键金句库

Hook
Skill 2.0 时代，来了。
MetaSkill 定义
MetaSkill 直接翻译就是元 Skill，也就是 Skill 的 Skill。
Skill 挑战
对于这些 Skill 如何把它更高效地进行检索，更高效地进行筛选组合，那这个是下一步 Agent 所面临的问题。
微内核
核心引擎只做最少的事，只进行编排调度和状态管理，其余的所有的能力全部是以插件的形式在用户态运行。
TOKEN 节省
整个项目帮助大家节省 60% - 80% 的 TOKEN 的成本。
Meta-Meta
创建 MetaSkill 的 MetaSkill 也是很有必要的。
二开
当前只支持英文，它是没有中文的，所以大家可以基于这个项目去做二次开发。
背景
他是原来华为盘古的一些核心的成员，出来去创的业，而且是今年刚刚创的业。
