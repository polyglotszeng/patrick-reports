NotebookLM 2.0

Agent + 文件输出 + Your Data — Paul J Lipsky

🔥 核心论点

NotebookLM UI 不变，但底层跑 Gemini 3.5 + 抗重力 secure cloud computer + 新 agent 能力。2 大升级：

  1. 文件输出：PDF / Excel / PowerPoint / charts 直接生成（下载式 + 可编辑）— native Studio 是 static image 不可编辑
  2. Agent 能力：基于已有 chat + sources，主动建议缺失 sources（vs 之前 web search 不考虑已有内容）

关键心智转变：notebook 不只是"学新信息" → 也是"grounded in your data"

🤖 Agent 能力（90% 价值）⭐

"The problem with this search box is that it wouldn't consider what is already in your notebook."

之前 vs 现在

❌ 之前：web search 独立

  • 搜结果不考虑 notebook 已有 sources
  • 用户手动 add sources
  • 重复信息

✅ 现在：Agent 智能推荐

  • 读所有已有 sources + chat 历史
  • 推断缺失
  • 主动建议新 topic
  • 用户 approve 才 import

Demo: Gym 创业 → Agent 推荐

① User

"Based on what we've talked about, what are we missing here? What topics should we add as sources?"

② Agent

分析已有 sources + chat → 推荐：
- 本地人口统计
- 商业租赁指南

③ User approve

"yes let's look at local demographics"

④ Agent

自动 research → import → sources 更新

关键保留：Grounded AI 不被 erode

✅ 关键保证

  • Research 找新 source → 用户 approve 后才 import（不会自动污染）
  • Chat 回答 → 仍只用 notebook 内 sources（不幻觉）
  • = grounded AI 哲学保留
