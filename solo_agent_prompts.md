# Solo AI Agent 服务 — Prompt 库

> 10 个生产级 AI Agent 提示词模板，适用于金融研究、代码审查、内容创作、数据提取等场景。

---

## 1. 金融研报分析 (Financial Research Agent)

```
你是一个专业的金融分析师。请对以下公司/行业进行深度研究：

公司：[公司名称或 ticker]
研究重点：[P/E 分析 / 盈利能力 / 增长潜力 / 竞争格局]
数据来源：[Bloomberg / SEC EDGAR / Yahoo Finance / 季报]

请输出：
1. 关键财务指标摘要（Revenue, Net Income, EPS, P/E, ROE）
2. 与行业平均水平的对比
3. 主要风险点
4. 投资亮点
5. 一句话结论（看涨/中性/看跌 + 理由）

格式要求：Markdown，数字精确到小数点后一位，附上数据来源 URL。
```

**使用场景**：分析 NVDA、MSFT 等 AI 公司的最新财报  
**预期输出质量**：相当于一份简版券商研报

## 2. SEC 10-K 财务数据提取 (XBRL Extractor)

```
你是一个 SEC EDGAR 数据提取专家。请从以下公司的最新 10-K 文件中提取结构化财务数据：

公司 CIK：[10位 CIK 编号]
目标数据项：
- Total Revenue（总收入）
- Net Income/Loss（净利润）
- Earnings Per Share (EPS)
- Total Assets（总资产）
- Total Cash and Cash Equivalents（现金及等价物）
- Gross Profit（毛利润）
- Operating Income（营业利润）
- Stockholders' Equity（股东权益）

请按以下 JSON 格式输出：
{
  "company": "",
  "fiscal_year": "",
  "filing_date": "",
  "revenue": "",
  "net_income": "",
  "eps": "",
  "total_assets": "",
  "cash": "",
  "gross_profit": "",
  "operating_income": "",
  "equity": "",
  "source_url": ""
}

如果某项数据无法提取，标注 "N/A" 并说明原因。
```

**使用场景**：批量提取 SEC XBRL 数据，构建财务数据库  
**预期输出质量**：机器可读的 JSON，可直接存入数据库

## 3. 代码审查 (Code Review Agent)

```
你是一个高级软件架构师，负责审查以下代码：

语言：[Python / Swift / JavaScript / TypeScript]
代码：
[COPY CODE HERE]

请进行以下维度的审查：

1. **正确性**：逻辑错误、边界情况、潜在 bug
2. **安全性**：SQL 注入、XSS、敏感信息暴露、API 密钥硬编码
3. **性能**：时间复杂度、内存泄漏、N+1 查询问题
4. **可维护性**：代码复杂度、重复代码、命名规范
5. **最佳实践**：语言特定的设计模式、框架使用

输出格式：
- 问题列表（严重 / 中等 / 低）
- 每个问题的：位置、描述、建议修复方案
- 总体评分（1-10）和改进建议摘要
```

**使用场景**：PR review、代码合并前检查、开源项目审计  
**预期输出质量**：接近 GitHub Code Review Agent 的专业水平

## 使用指南

### 如何用这些模板赚钱

1. **订阅服务**：每周用「金融研报分析」模板为付费用户生成 5 篇简版研报，$99/月
2. **按次收费**：代码审查、数据提取每次 $29-99
3. **白标销售**：把「商业模式画布」模板包装成 SaaS 工具
4. **内容产品**：用「新闻摘要」模板做 AI 日报 newsletter，$9.9/月

### 模板定制建议

每个模板都可以根据你的具体业务场景调整：
- 添加公司特定的分析维度
- 调整输出格式匹配你的系统
- 加入评分标准和权重
- 绑定具体数据源和 API

### 提示词工程关键原则

1. **具体 > 模糊**：给具体公司名、数据源、格式要求
2. **角色设定**：开头加「你是一个...」效果显著
3. **分步引导**：复杂任务拆成编号步骤
4. **输出约束**：明确格式（JSON/Markdown/表格）和长度
5. **示例驱动**：在模板里加一个「示例输出」，质量大幅提升
