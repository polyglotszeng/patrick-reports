技术原理
组件 说明
Profile 隔离 每个 profile 独立 config.yaml + .env，HERMES_HOME 隔离，credentials 不泄露
CLI 别名 fast → hermes -p fast，零延迟切换模型
Cron 模型指定 --model minimax/MiniMax-M2.7-highspeed 精确指定每 job 模型
delegate_task 路由 Orchestrator subagent 可根据任务类型 spawn 指定 profile 的 agent
凭证管理 auth.json 统一管理所有 provider credentials，profile 共享认证状态
