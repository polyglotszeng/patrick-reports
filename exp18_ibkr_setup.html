③ Python 3.9+

Mac 自带, 验证 python3 --version

端口 & API 配置 (TWS 里)

TWS → Edit → Global Configuration → API → Settings
  ☑ Enable ActiveX and Socket Clients
  ☑ Allow connections from localhost only
  Socket port: 7497  (paper 默认)
  Trusted IP:  127.0.0.1
  Client ID:   18    (任意 1-32, 脚本默认 18)

3. 5 Falsification 框架 核心

跑完自动检查 5 项 — 任何 1 项 ✗ 都判数据不可用

  1. F1 公司数 = 30 — 实际 30 个 ticker, 少一个 = 重抓
  2. F2 季度数 = 16 — 每公司 16 财年季度, min/max 都 16
  3. F3 NaN < 5% — OHLC 5 列 NaN 率均值 < 5%
  4. F4 价格 sanity — 全 close > 0, 无 ±200% 极端跳变
  5. F5 4 宏观列存在 — VIX/NDX/10Y/USD_CNY 都有且 NaN < 50%

检查报告: exp18_real_data/sanity_check_report.json (JSON, 5 项 pass/fail)

5. 错误速查 + 下一步

常见错误

报错原因修法
IBKR connection timeout TWS 没开 / 端口错 lsof -iTCP:7497 确认 LISTEN
FRED Invalid API key key 错 / 没点邮件激活 echo $FRED_API_KEY 重新登 FRED 查
yfinance 429 限流 (周末凌晨几乎不触发) 脚本内 sleep(0.5) 已保护, 改 1.0
ib_insync ImportError 没装 / 装到别的 Python pip3 install --user ib_insync 重装
