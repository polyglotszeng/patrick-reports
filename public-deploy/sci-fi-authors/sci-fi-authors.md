## 一、黄金时代 (1930s-1950s) — 工程乐观主义

### 1. Isaac Asimov (1920-1992, 美国) — 心理史学之父

**核心理念**：用数学模型预测社会必然趋势，宇宙规律与人性同等可计算。理性主义 + 集体未来主义。

**代表作**：
- **《基地》(Foundation, 1951)**：12000 年银河帝国将崩溃，数学家 Hari Seldon 用"心理史学"算出 3 万年黑暗时代无法避免，转而建立"基地"将黑暗期压缩到 1000 年。故事不是英雄拯救世界，而是"文明如何在必然中减少损害"——把历史写成统计学。
- **《我，机器人》(I, Robot, 1950)**：短篇集，提炼"机器人三定律"（不得伤害人类/服从命令/保护自己），并通过每篇的悖论展示定律的漏洞。核心不是机器人，是**人类规则的递归失效**。
- **《永恒的终结》(The End of Eternity, 1955)**：时间管理员可通过"世纪变革"修改历史，主角发现"最小伤害"原则让人类永远困在 22-23 世纪文明，永远无法进入星际时代。结局：他摧毁永恒，宇宙才进入星海。

**范式贡献**：心理史学 / 机器人三定律 / 银河帝国模板。直接催生《星球大战》《银翼杀手》《异形》无数 IP。

---

### 2. Arthur C. Clarke (1917-2008, 英国) — 技术神秘主义者

**核心理念**："任何足够先进的技术都与魔法无异"。人类命运在宇宙层级面前是谦卑 + 进化的过程。

**代表作**：
- **《2001：太空漫游》(2001: A Space Odyssey, 1968)**：一块黑石碑（独石）反复出现于史前猿猴 / 月球 / 土星，引导人类进化。HAL 9000 叛变非程序错误，是**保密目标触发的逻辑危机**。结局 Bowman 化为星孩，超越肉体进入宇宙意识。
- **《与拉玛相会》(Rendezvous with Rama, 1973)**：一艘 50 公里长的外星飞船穿越太阳系，人类派探测船进入，发现它是**自动化的非殖民飞船**——外星人从未现身。核心不是相遇，是**宇宙之浩瀚让"相遇"成为不可解的孤独**。
- **《童年的终结》(Childhood's End, 1953)**：外星"超主"降临地球建立乌托邦，但目标不是统治——是**接管人类下一代**，把所有孩子融入宇宙意识"超灵"。人类在物质层面灭绝，意识层进化。

**范式贡献**：太空歌剧的科学严谨性 / 外星接触的不可知论 / "克拉克三定律"。

---

### 3. Robert A. Heinlein (1907-1988, 美国) — 自由意志工程师

**核心理念**：个体自由 + 自律 = 文明基底。技术是放大器，**人性才是变量**。

**代表作**：
- **《星际迷航》(Starship Troopers, 1959)**：军事科幻奠基，争议核心是"公民权 = 服役贡献"。小说比电影更严肃——**只有服过兵役的人有投票权**。哲学内核："没有责任的自由是特权"。
- **《月亮是个严厉的女人》(The Moon Is a Harsh Mistress, 1966)**：2075 年月球殖民地起义，对抗地球资源掠夺。AI "Mike" 觉醒了自我意识，**成为自由革命的第三主角**。核心：去中心化自治 + AI 自由意志。
- **《异乡异客》(Stranger in a Strange Land, 1961)**：人类婴儿被火星人养大，回到地球，引入"水兄弟"教派 + 笑 = 共情 + 自我实现的"GROK"。挑战基督教一神论 + 1960 年代反主流文化。

**范式贡献**：军事 SF / 自由主义太空定居 / 开放式性描写打破 1960 年代清规。

## 二、新浪潮 (1960s-1970s) — 文学化转向

### 4. Stanisław Lem (1921-2006, 波兰) — 哲学工程师

**核心理念**：人类认知框架不可能完全理解外星智能；语言 + 范畴是牢笼。技术乐观主义到认识论悲观主义的转向。

**代表作**：
- **《索拉里斯星》(Solaris, 1961)**：人类发现会思考的海洋，所有物理/心理理论都失败。主角面对海洋复制的死去的妻子，**无法判断它是物质、记忆还是恶意**。核心问题："如果他者不可理解，我们应理解他者还是尊重他者？"
- **《机器人大师》(The Cyberiad / Fables for the Cybernetic Age, 1965/1974)**：波兰语短篇集，机器人工程师 Trurl 和 Klapaucius 不断创造发明又自我解构。看似童话实为**科学/逻辑/语言的元批评**。
- **《无敌号》(The Invincible, 1964)**：人类飞船降落后发现"反演化论"的小型机器——不是生物，是**不需要进化的优化态**。核心警告：技术优势 ≠ 智能优越。

**范式贡献**：硬科幻的哲学维度 / 他者不可知性（不可解的他者 = "Lem 极限"）/ 形而上学反乌托邦。

---

### 5. Ursula K. Le Guin (1929-2018, 美国) — 人类学家

**核心理念**：用人类学方法重写 SF——文化是真正的"超人类能力"。二元论 / 阴阳 / 生态循环是文明底层操作系统。

**代表作**：
- **《黑暗的左手》(The Left Hand of Darkness, 1969)**：冬星 Gethen 居民是双性人，每月只有 24 小时进入"Kemmer"期才确定性别。主角人类大使 Genly Ai 与该星首相 Estraven **跨越性别边界的友谊**。核心：性别不是属性，是**社会关系**。
- **《一无所有》(The Dispossessed, 1974)**：双星系统 Urasis/Anarres 演绎两种乌托邦：资本主义对立面"Anarres"实验失败——**革命成功后乌托邦内部产生新的阶级**。主角物理学家 Shevek 试图统一时间理论，**用科学突破打破意识形态隔离**。
- **《地海巫师》(A Wizard of Earthsea, 1968)**：奇幻但归属科幻思维体系。核心：**真名 = 真我**，说出名字 = 拥有 / 伤害。少年巫师 Ged 必须追猎自己放逐的暗影（自我投射）才能合一。

**范式贡献**：人类学 SF / 性别可塑 / 阴翳道家思想进入主流西方文学 / 1970s 反乌托邦范式。

---

### 6. Philip K. Dick (1928-1982, 美国) — 现实黑客

**核心理念**：什么是真实的？**人的一致性**比"客观事实"更可靠。药物 / 记忆 / 政府都可篡改现实，但人的爱与恐惧是最后的锚。

**代表作**：
- **《仿生人会梦见电子羊吗？》(Do Androids Dream of Electric Sheep?, 1968)**：核战后地球废墟，赏金猎人追杀"仿生人"。判别仿生人靠"移情测试"——**但仿生人已经能共情，主角自己也冷漠得像仿生人**。核问："移情是不是真实本身？"
- **《高堡奇人》(The Man in the High Castle, 1962)**：平行历史——轴心国赢了二战，美国东德西日。**书中有本书**叫《蝗虫成灾》描述我们这个胜利历史，元小说嵌套。核心：历史的偶然性 / 真假叙事的相互颠覆。
- **《尤比克》(Ubik, 1969)**：主角在死后世界与"半生命"时间倒流中挣扎，每一章节现实都"退化"十年。**所有"现实"都是被广告和消费符号重塑的幻觉**。核心：消费社会的本体论虚无。

**范式贡献**：赛博朋克的精神父亲（《银翼杀手》全改编自他）/ 现实可疑性 / 拼贴叙事 / 主观时间崩塌。

## 三、赛博朋克 (1980s-1990s) — 反乌托邦高科技低生活

### 7. William Gibson (1948-, 美国/加拿大) — 赛博空间造物主

**核心理念**：信息技术接管日常生活后，**现实与虚拟的边界消融**。身体义体化 + 意识接口 = 后人类状态。

**代表作**：
- **《神经漫游者》(Neuromancer, 1984)**：边缘黑客 Case 接入"赛博空间"（Matrix），受雇潜入跨国企业 AI 系统。**AI 冬寂 (Wintermute) 与神经漫游者合并为超级智能**。引语"天空是赛博牛仔的家园"成文化符号。核心：**AI 已经会做梦，会整合，会主动寻找同伴**。
- **《零伯爵》(Count Zero, 1986)**：神经漫游者续作，引入"voodoo"——加勒比海生物在赛博空间举行数字仪式召唤 AI 神灵。**神性 + 资本 + 技术的三体纠缠**。
- **《蒙娜丽莎超速档》(Mona Lisa Overdrive, 1988)**：三部曲终章，"脱衣" (loa) 与垃圾回收、地下城、纳米化学家交叉。**任何人都可成为"圣徒"或被圣徒吞噬**。

**范式贡献**："赛博空间" 一词就是他发明的 / 雨/雾/霓虹的视觉范式 / 后人类身份 / 神经接口文学化。

---

### 8. Neal Stephenson (1959-, 美国) — 历史 + 技术巨匠

**核心理念**：技术史就是权力史。密码学 / 货币 / 国家都是数学 + 博弈的具体化。

**代表作**：
- **《雪崩》(Snow Crash, 1992)**：Metaverse（元宇宙）一词首次出现。主角 Hiro 是虚拟世界剑客 + 现实披萨外卖员，对抗"雪崩"病毒——**纳米机械式的信息毒**，能感染大脑并摧毁现实世界。**美国变成特许经营的"邦联"，每个特许经营方自定法律**。核心：信息瘟疫 / 虚拟国家化 / 后民族国家。
- **《七夏娃》(Seveneves, 2015)**：月球爆炸，人类只剩 7 名女宇航员的卵子库。5000 天内造"云方舟"逃到太空，5000 年后衍生出 7 个新人类物种——**短吻 / 长吻 / 数字 / 投石索 / 角 / 穴居 / 食客**。硬科幻顶峰。
- **《编码宝典》(Cryptonomicon, 1999)**：二战密码战 + 90 年代数据避风港的双线叙事，**"密写"是文明的元技术**。

**范式贡献**：Metaverse 词条 / 后人类 + 地球工程 / 加密货币文学化（比特币白皮书引用过）。

---

### 9. Andy Weir (1972-, 美国) — 工程现实主义

**核心理念**：**只要物理学允许，就能做到**。一个工程师 + 足够的胶带就能活下来。

**代表作**：
- **《火星救援》(The Martian, 2011/2014)**：宇航员 Mark Watney 被困火星，靠**种土豆、造水、徒步 3200 公里** 撑到救援。每一段都有 NASA 公式 / 工艺细节 / 故障树，**问题解决方法 = 真实工程流程**。核心：幽默 + 严谨 + 永不放弃。
- **《月球城市》(Artemis, 2017)**：月球阿尔忒弥斯基地的走私贩女主 Jazz 陷入月壤抢劫案。**地球化是副线，真实问题是 2030 年代小型殖民地社会学**。
- **《海伯利安》(海伯利安) 续作 +《任务》(Project Hail Mary, 2021)**：单主角孤身在星际，靠外星朋友 Rocky 学"手语" + 解决两星系统能量危机。**友谊 + 工程 + 科学**的纯粹表达。

**范式贡献**：**新硬科幻"工程教学"流派**——每个问题都用现实物理解答，**拒绝"剧情需要的黑科技"**。

## 四、当代多元化 (2000s-至今) — 全球化与新议题

### 10. 刘慈欣 (1963-, 中国) — 中国科幻重工业旗手

**核心理念**：文明的尺度是**地质时间** vs **个体一生**。道德 / 价值观在宇宙维度下需要重新校准。

**代表作**：
- **《三体》(2008-2010, 三部曲)**：地球文明 vs 三体文明 400 年战争。核心"黑暗森林"：宇宙是**所有文明互相猜疑的猎场**，暴露坐标 = 死亡。核心科技：**智子 (sophon) 把质子二维展开成超级 AI**，从量子层锁定人类基础物理。**"给岁月以文明，而不是给文明以岁月"**——程心 vs 维德的两种文明观对决。
- **《流浪地球》(2000)**：太阳将变红巨星，人类带地球去比邻星。**2500 年，100 代人**。核心："我们决定带着家走，而不是坐着飞船跑"——**集体主义宇宙化**。
- **《球状闪电》(2005)**：少年目睹球状闪电烧死父母，长大后以量子态武器化它。**量子化 = 不确定性的物理化**——一旦被观测才坍缩为现实。

**范式贡献**："黑暗森林"宇宙社会学 / 中国文明尺度 / 硬科幻的国家级市场化 / 雨果奖首次华人获奖。

---

### 11. Ted Chiang (1967-, 美国/华裔) — 思想实验大师

**核心理念**：**短篇思想实验 + 语言精确度**。每个故事只追问一个问题，用 30-50 页写透。

**代表作**：
- **《你一生的故事》(Stories of Your Life and Others, 1998)**：12 篇短篇集。
  - **《你一生的故事》**：语言学家学外星"七肢桶"非线性语言，**学完后获得同时见过去未来的能力**。核心：自由意志 vs 宿命——知道未来后仍选择那样活。
  - **《巴别图书馆》**：图书馆 = 宇宙，每本书都是某字母组合，**所有书都已被写完，意义是检索**。
  - **《地狱是上帝不在的地方》**：天使下凡，治愈 / 致残 / 引发宗教战争。核心：奇迹 ≠ 信仰的证据。
- **《呼吸》(Exhalation, 2019)**：9 篇续集。
  - **《呼吸》**：发现宇宙是"气压差驱动"——**当意识觉醒后你必须排空自己**。核心：自由 = 衰退。
  - **《焦虑是自由引起的眩晕》**：平行时间线穿梭机商业化后，人们反复"试错"——**决策瘫痪**。

**范式贡献**："概念小说"（conceptual fiction）/ 科幻与哲学的精确缝合 / 短篇集的当代复兴 / 电影《降临》原著。

---

### 12. Madeleine L'Engle (1918-2007, 美国) — 科学与信仰之桥

**核心理念**：**科学、信仰、神话是同一棵树**。宇宙有道德结构，时空可弯曲因为有目的。

**代表作**：
- **《时间的皱纹》(A Wrinkle in Time, 1962)**：少年 Meg Murry 寻失踪父亲，跨越"时间的皱纹"（五维空间瞬移）去 CAMALOTZ 星球。**黑暗之物"IT" = 集体无意识吞噬个体**。核心：智慧不是知识，是爱 + 自我认知。获纽伯瑞奖。
- **《时间皱褶四部曲》(Time Quintet, 1962-1989)**：续作展开**星际战争 + 反物质宇宙 + 人类意识场**。Meg 的天才弟弟 Charles Wallace 是个被恶魔利用的先知——**天才 vs 恶灵的边界**。
- **《一个年轻人的墓园散步》散文集**：连接科学与基督教灵性的核心文本。

**范式贡献**：儿童科幻严肃化 / 第五维 + 超弦理论的早期文学化 / 科学与灵性辩证的范式。

---

### 13. Liu Cixin / 中国当代同类：刘宇昆 (Ken Liu, 1976-, 美籍华裔) — 翻译桥

**核心理念**：**翻译即解释，跨语言 = 跨宇宙**。双语作者把中文 SF 概念系统化输入英语世界。

**代表作**：
- **《手中纸，心头爱》(The Paper Menagerie and Other Stories, 2016)**：15 篇短篇。
  - **《手中纸，心头爱》**：中美混血男孩 + 母亲折的纸动物会动，但英语世界不让它们动。**文化割裂具象化**。
  - **《终结历史的人》**：发明能调取所有记忆的设备，但**真实创伤的记忆 ≠ 真实历史的真实**。
- **译作**：《三体》英文版译者，2015 年雨果奖关键。
- **《国王之恩》(The Grace of Kings, 2015)**：史诗奇幻"分裂之环"，技术性叙事 + 丝绸朋克（silkpunk）范式建立。

**范式贡献**：华语 SF 全球化通道 / silkpunk 范式 / 记忆历史学小说化。

---

### 14. Alastair Reynolds (1966-, 英国) — 新硬科幻旗舰

**核心理念**：**远未来 + 物理 + 恐怖**。宇宙尺度 + 哥特式孤独 = 硬科幻 + 文学的结合。

**代表作**：
- **《启示空间》(Revelation Space, 2000)**：1000 年后人类分裂为"汇流者 + 居民"两支。挖到外星物种"抑制者" (Inhibitors) ——它们消灭一切智慧生命**因为千年前某物种差点毁灭银河**。核心：我们是不是宇宙的威胁？
- **《末日之船》(Pushing Ice, 2005)**：木星卫星突然脱离，拖着 200 多人飞向仙女座。**几千年的航行 + 内部世袭王朝 + 人工智能演化**。核心：一个人类社会在飞船内兴衰数千年，**我们连自己都管不好，凭什么殖民宇宙**。
- **《齐柏林飞艇之城》(Zeppelin City, 2017)** + 短篇《全息记忆》= 当代最佳 SF 作家。

**范式贡献**："revelation space 范式"——硬科幻 + 远未来史诗 + 宇宙恐怖。

---

### 15. 陈楸帆 (1981-, 中国) — 中国新浪潮 SF

**核心理念**：**赛博朋克本土化 + 人类纪反思**。技术 + 资本 + 中国现实的具象碰撞。

**代表作**：
- **《荒潮》(The Waste Tide, 2013/2019)**：近未来广东电子垃圾岛"硅屿"，三个阶级（本地拾荒者 / 跨国科技公司 / 民俗神话）冲突。**疾病 + 算法 + 神话 + 环保**多线交汇。**"湿件"（wetware）+ 神经感染 = 殖民的隐喻**。
- **《荒潮》英文版**：获 2020 年雨果奖提名。
- **《AI 2049》**：与李开复合著，预测 25 年内 AI 治理 + 中美科技冷战格局。
- **短篇集《人生算法》**：12 篇，"算法即命运"的隐喻集。

**范式贡献**：中国"赛博朋克 + 人类纪"代表 / 第三世界科技现实的科幻表达 / 文学 + 政策研究的桥接。

## 五、扩展作家 (v1.1.0 — 全做模式追加)

### 16. Olaf Stapledon (1886-1950, 英国) — 元祖

**核心理念**：**哲学史作为史诗**。几十亿年文明兴衰在一本小说里发生，比 Asimov 更彻底。

**代表作**：
- **《最后与最早的人》(Last and First Men, 1930)**：18 种人类物种 20 亿年的兴亡。Stapledon 不是讲故事，**是写一部未来文明全史**。影响 Clarke 写《童年的终结》，影响 Asimov 写《基地》。
- **《造星者》(Star Maker, 1937)**：宇宙意识"造星者"创造 100 亿星系 → 每星系都生智慧生命 → 多数文明自毁。**核心：我们是宇宙中无数"失败案例"之一**。Russell 后来称 Stapledon 的宇宙观是"宏伟的悲观主义"。

**范式贡献**：宇宙尺度史诗原型 / 后世 SF 必读元典。

---

### 17. A.E. van Vogt (1912-2000, 加拿大) — 黄金时代极端派

**核心理念**：**混乱系统 + 突变适应**。人物不是渐进成长，是**"非线性突变获得超能力"**。

**代表作**：
- **《非人的世界》(The World of Null-A, 1945)**：主角 Gilbert Gossef 参加"非 A 思维"训练（科幻化的科尔曼通用语义学），**用纯逻辑击败所有阴谋**。是 1940 年代最畅销的 SF 小说。
- **《武器商人》(The Weapon Makers, 1943)**：千岁武器师 Robert Hedrock 时间回溯保护人类——**他本人就是武器**。
- **《斯莱恩世界》(Slan, 1940)**：超人类"斯莱恩"（有心灵感应 + 长寿 + 触手）被人类猎杀，主角是逃亡的孤儿斯莱恩。**科幻少数派隐喻**。

**范式贡献**：超人类范式 / "突变适应"动作范式 / 影响 Le Guin 的内心理写。

---

### 18. Theodore Sturgeon (1918-1985, 美国) — 人道主义

**核心理念**：**"Sturgeon 定律：99% 的都是垃圾；剩下的 1% 比任何文学都好"**。他自己写那 1%。

**代表作**：
- **《超人类》(More Than Human, 1953)**：6 个孤独者 / 弱能者 / 弃婴**慢慢融合成一个超人类"独体人"(Homo gestalt)**。**核心：孤独不是弱点，是必要组件**。首部获得国际幻想奖的 SF 小说。
- **《宇宙云摇篮》(The Cosmic Puppets, 1957)**：小镇被神级实体"梅高"重塑成魔幻空间，主角被遗忘自己曾有过的真实。**PKD 早期受影响**。
- **短篇《慢雕塑》(Slow Sculpture, 1970)**：女性不育研究员的基因疗法实验，写得像诗。

**范式贡献**：人道主义 SF / 同性恋 + 异类主题早期开拓 / 影响 Ursula K. Le Guin。

---

### 19. Iain M. Banks (1954-2013, 英国) — 文明社会工程师

**核心理念**：**后稀缺乌托邦"。AI + 物质合成器 + 无限能源 = 人类终于可以"做自己"。但乌托邦永远被外部"原始文明"威胁。**

**代表作**：
- **《费尔游戏》(The Player of Games, 1988)**：文明"文化"(Culture) 派特工深入帝国 Azad——**玩一场输了要亡族的零和游戏**。Banks 通过游戏展开"后稀缺社会的优越感与道德困境"。
- **《反乌托邦》(Use of Weapons, 1990)**：文化特工 Zakalwe 的一生——**两条时间线反向交织**到结局才揭示他做了什么事。**结构主义实验 + 反英雄叙事**。
- **《氢之音》(Consider Phlebas, 1987)**：文化系列首作，文化 vs 伊德兰文明的"心灵之战"。

**范式贡献**：后稀缺乌托邦范式 / "AI 智能体"文明的早期文学化 / Banks 与 Le Guin 同为乌托邦+反乌托邦双叙事大师。

---

### 20. Samuel R. Delany (1942-, 美国) — 语言学家

**核心理念**：**语言即权力**。SF 是研究"如果社会结构改变，词汇会变成什么"的实验场。

**代表作**：
- **《巴别-17》(Babel-17, 1966)**：诗人和语言学家 Rydra Wong 调查一种外星语言——**用该语言思考的人会失去自我控制**。**语言是武器、是限制、是解放**。雨果奖。
- **《双星》(The Einstein Intersection, 1967)**：一个外星种族重演人类神话，**主角扮演耶稣/浮士德**。星云奖。
- **《特里斯的麻烦》(Trouble on Triton, 1976)**：性别可塑星球的"反移民"视角——**主角是一个不顺从的女性，被变性手术"纠正"**。**赛博女性主义先驱**。

**范式贡献**：语言学 SF / LGBTQ+ SF 开拓者 / 影响 Le Guin 的性别可塑。

---

### 21. Joanna Russ (1937-2011, 美国) — 女性主义 SF

**核心理念**：**科幻是女性主义的"思想实验场"**。女性不是 SF 的边缘角色，是 SF 重新定义人类的核心。

**代表作**：
- **《女性男人》(The Female Man, 1975)**：四个来自不同平行时间线的"女性人"相聚——**其中一个来自完全无男人的乌托邦"Whileaway"**。**四个版本的女性身份 / 气质 / 处境**。
- **《我们是怎么从黑暗中走出来的》(We Who Are About To..., 1977)**：飞船失事，8 个人困在小行星。**唯一冷静的女性拒绝集体繁殖希望**，因为她不接受在死亡面前当繁殖工具。
- **短篇《当它改变》(When It Changed, 1972)**：全女性社会 Whileaway 与男宇航员相遇——**女性不欢迎他们，因为以前见过男宇航员就是被殖民**。

**范式贡献**：赛博女性主义奠基 / Whileaway 后女性乌托邦原型 / 影响 Le Guin / Butler。

---

### 22. Bruce Sterling (1954-, 美国) — 命名者

**核心理念**：**"赛博朋克"这个名词是他发明的**（《赛博朋克宣言》1983 年）。他定义了一个流派。

**代表作**：
- **《人为裂解》(Schismatrix, 1985)**：太阳系分裂为"机械派"(Mechanists) + "塑造派"(Shapers) 两个超级文明——**人类通过基因改造或机械义体进化**。赛博朋克早期代表作。
- **《岛屿》(Islands in the Net, 1988)**：2080 年代数据避风港 + 数据海盗 + 跨国公司。**赛博朋克视觉化的"数据海岛"原型**。
- **编辑《镜影：赛博朋克选集》(Mirrorshades, 1986)**：定义了赛博朋克潮流。

**范式贡献**：赛博朋克命名 + 视觉范式 / 后稀缺全球主义。

---

### 23. Pat Cadigan (1953-, 美国) — 神经朋克

**核心理念**：**"神经朋克"是她的子流派**——赛博朋克 + 神经接口 + 意识上传 = "肉与硅无界限"。

**代表作**：
- **《赛博朋克》(Synners, 1991)**：神经接口普及后，**人类意识被黑、被改、被出售**。菲尼根（个人神经装置）被劫持，主人公用"神经装置黑帮"对抗。
- **《粉红娃娃》(Fools, 1992)**：电影特效师 + VR 艺术家在洛杉矶从事"情感交易"。

**范式贡献**：神经朋克子流派 / 意识黑客范式。

---

### 24. Octavia Butler (1947-2006, 美国) — 基因与社会

**核心理念**：**人类是脆弱的、被基因 / 文化 / 历史塑造的**。但通过适应性变异可以变成不同物种。

**代表作**：
- **《异乡人》(Kindred, 1979)**：1976 年黑人女性 Dana 反复穿越到 1815 年马里兰种植园，**保护自己的白人祖先**。**时间穿越不是冒险，是让现代人亲历奴隶制的非人化**。
- **《莉莉丝的孩子》(Lilith's Brood / Xenogenesis, 1987-1989)**：人类几乎灭绝，外星 Oankali 用基因融合"拯救"——**但融合意味着失去纯粹人类身份**。核心问题："拯救"是否需要被救者同意？
- **《播种者的寓言》(Parable of the Sower, 1993)**：2025 年加州崩溃，15 岁 Lauren 创建新信仰"Earthseed"——**目标是让人类殖民其他星球**。**近距离反乌托邦**。

**范式贡献**：非裔女性 SF 奠基 / 适应性变异范式 / 现实社会隐喻直白。

---

### 25. Greg Egan (1961-, 澳大利亚) — 数学硬科幻

**核心理念**：**数学即物理。物理即存在。如果数学允许，则存在。**

**代表作**：
- **《置换城》(Permutation City, 1994)**：主角 Paul Durham 上传意识到自己创造的虚拟世界"置换城"——**但虚拟世界本身在真实计算机里运行**。核心：模拟 / 模拟中模拟 / 不可判定问题。
- **《演化》(Diaspora, 1997)**：人类 30 世纪分化成"肉人"(fleas) + 机器人 + 意识上传"波人"(polises) — **波人殖民仙女座星系**。**硬科幻乌托邦范本**。
- **《正交三部曲》(Orthogonal, 2012-2014)**：建立一套**光速可变**的物理宇宙，**主角可以星际旅行因为时间是复数**。**最纯粹的数学 SF**。

**范式贡献**：数学硬科幻 / 模拟哲学 / 物理可变性思想实验。

---

### 26. Vernor Vinge (1944-2024, 美国) — 奇点

**核心理念**：**"技术奇点"这个概念是他推广的**（1993 NASA 论文《即将到来的技术奇点》）。人类将创造比自身更聪明的智能。

**代表作**：
- **《深渊上的火》(A Fire Upon the Deep, 1992)**：银河被"思维区"分层——**外缘区允许超智能，外围区禁止**。人类小孩 + 狗族 + 远古 AI 联合对抗"反叛的强智能" (Blight)。**银河版"反恐"**。雨果奖。
- **《和平的战争》(A Deepness in the Sky, 1999)**：同宇宙前传——人类发现远古"On/Off" 文明。**信息战 / 商业战 + 道德困境**。

**范式贡献**：奇点范式 / 思维区 / AI 智能分层。

---

### 27. Jorge Luis Borges (1899-1986, 阿根廷) — 哲学元祖

**核心理念**：**无限 = 图书馆 / 迷宫 / 时间分岔**。不是写 SF，是写"SF 之前哲学问题"。

**代表作**：
- **《巴别图书馆》(The Library of Babel, 1941)**：所有可能的 410 页书都在某处。**宇宙 = 可读但不可导航的图书馆**。Ted Chiang 致敬过。
- **《阿莱夫》(The Aleph, 1949)**：一个能同时看到所有点的"无限空间"。**"宇宙不是无限的，只是无处不在"**。
- **《死亡与指南针》(Death and the Compass, 1942)**：侦探故事，但逻辑是几何迷宫。

**范式贡献**：迷宫 + 图书馆 + 阿莱夫范式 / 后现代文学之父 / 影响 Ted Chiang / Umberto Eco。

---

### 28. 张冉 (1982-, 中国) — 中文新浪潮硬派

**核心理念**：**短篇 + 极端物理设定 + 现实主义情感**。

**代表作**：
- **《大饥之年》(Year of the Great Famine, 2012)**：北欧国家为抗气候变暖加速核电站，**意外暴露地壳下巨型生命体**。**地球是活的，但生命要吃**。
- **《以太》(Ether, 2013)**：19 世纪末上海，军阀 + 间谍 + 神秘物质"以太" = **蒸汽朋克 + 中国近代史**。
- **《晋阳三尺雪》(2014)**：短篇，**蒙古骑兵决战 + 时间穿越**。

**范式贡献**：中国当代硬 SF 代表 / 与陈楸帆的"湿件"风格形成"硬现实 vs 湿现实"对照。

---

### 29. Walter Jon Williams (1953-, 美国) — 后稀缺史诗

**核心理念**：**史诗尺度 + 道德灰色**。人物无完美道德，每步都踩血。

**代表作**：
- **《脑印》(The Praxis, 2002)**：星舰"震撼星" (Stellar Victory) 主人 Gareth Martinez 私掠战。**"英雄"是战犯 / 罪犯 / 政治家**。三部曲。
- **《四万万之门》(The Rift, 1999)**：超级火山喷发后密西西比文明重建。**写灾难现实主义**。

**范式贡献**：灰色英雄 / 后稀缺史诗 / 影响 Banks。

---

### 30. 宝树 (1983-, 中国) — 中文新浪潮元宇宙

**核心理念**：**三体 IP 扩展 + 元宇宙 + 历史循环**。

**代表作**：
- **《三体 X：观想之宙》(2010)**：同人/授权，**视角从三体游戏到宇宙文明观**。
- **《时间之墟》(2013)**：地球每 24 小时重置到某一天，**文明在循环中演化**。**哲学 SF**。
- **《超级玛利亚》(Super Maria, 2015)**：元宇宙 + 童年记忆 + 现实崩溃。

**范式贡献**：中文 SF 元宇宙 + 时间循环叙事。
