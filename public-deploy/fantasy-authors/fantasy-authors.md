## 🌑 经典奠基 (1900-1955)

### 1. Lord Dunsany (1878-1957, 爱尔兰/英国) — 现代奇幻之父
**核心理念**：把神话、梦境、东方主义熔于一炉，开创"第二世界"文学范式，影响托尔金、Le Guin、C.S. Lewis。
- **《裴伽纳诸神》(1905)**：现代奇幻第一本"造神小说"——一群永生但无用的人开会决定事物的形状
- **《精灵国王的女儿》(1924)**：村庄英雄穿越边界带回精灵公主，但代价是"没有边界的地方没有故事"
- **范式贡献**：现代奇幻叙事模式 / 短篇诗化 / "第二世界"文学概念

### 2. E.R. Eddison (1882-1945, 英国) — 恶棍式英雄
**核心理念**：拒绝托尔金的"善必胜"，建立复杂的反派美学。贵族政治 + 史诗战争。
- **《蠕虫奥博洛斯》(1922)**：Lord Juss 远征 Witchland 与 Lord Gro 决战，故事结尾是循环——所有英雄回到原点。"Ouroboros" = 无限循环
- **范式贡献**：反派美学 / 贵族政治奇幻 / 影响了 Michael Moorcock 和 grimdark

### 3. Robert E. Howard (1906-1936, 美国) — 剑与魔法的工业化
**核心理念**：创造了 Conan 等系列英雄，**把奇幻变成 pulp magazine 的高量产商品**。1936 年 30 岁自杀。
- **《柯南》(1932-1936)**：野蛮人 Cimmeria 战士在 Hyborian Age 历险，最终成为 Aquilonia 国王。力量 + 性感 + 反文明
- **《所罗门·凯恩》(1928)**：清教徒复仇者，混合恐怖 + 剑斗
- **范式贡献**：Sword and Sorcery 子类型奠基 / 柯南宇宙扩展至今 / 影响了 WoW 魔兽世界

## ⚔️ 史诗回归 (1954-1977)

### 4. J.R.R. Tolkien (1892-1973, 英国) — 现代奇幻双柱之一
**核心理念**：把中土世界构建成**有完整语言、历史、地理、神话的神造世界**。语言学教授的二次创造 = 文学的创世论。
- **《霍比特人》(1937)**：Bilbo Baggins 与 13 个矮人远征孤山，偶然获得魔戒 + 咕噜的猜谜
- **《魔戒》三部曲(1954-1955)**：Frodo 携带至尊戒前往末日火山销毁。9 名护戒使者 = 自由民族联盟，Sauron = 极权诱惑
- **《精灵宝钻》(1977, 子辑)**：精灵与人类第一纪元的完整编年史
- **范式贡献**：完整 secondary world / 精灵语 Quenya/Sindarin / 影响了几乎所有现代奇幻

### 5. Ursula K. Le Guin (1929-2018, 美国) — 双柱之二 + 阴翳道家
**核心理念**：用道家"无为"哲学反托尔金的男性中心，**奇幻不必有战争和龙**。
- **《地海》系列(1968-2001)**：群岛 Earthsea, 少年巫师 Ged 必须追猎自己放逐的暗影 (自我投射) 才能合一。"真名 = 真我"
- **范式贡献**：阴翳道家范式 / 短篇深刻化 / 反史诗反二元 / 影响了 N.K. Jemisin

### 6. Mervyn Peake (1911-1968, 英国) — 哥特奇幻巅峰
**核心理念**：把家族城堡变成一个有机体，**主角是反英雄的室内设计师**。
- **《歌门鬼城》(Gormenghast, 1950)**：Titus Groan 出生在巨大石城，整座城堡的仪式、墙缝、屋顶都是"角色"
- **范式贡献**：哥特奇幻 / 文学化奇幻 / 影响了 grimdark / Gene Wolfe 借鉴

## 🌍 多元爆发 (1980-2007)

### 7. Terry Pratchett (1948-2015, 英国) — Discworld 讽喻大师
**核心理念**：32 部 Discworld 用**喜剧 + 政治讽喻 + 哲学**，把奇幻变成"分析时代"工具。
- **《碟形世界》(1983-2015) 32 部**：平顶圆盘骑在 4 头大象上, 大象站在太空龟背上
- **《好兆头》(1990, 与 Gaiman 合著)**：天使 + 恶魔阻止末日，喜剧讽喻
- **范式贡献**：讽喻奇幻 / 反英雄主角 / 影响了 Douglas Adams

### 8. George R.R. Martin (1948-, 美国) — grimdark 旗手
**核心理念**：继承霍华德 + 皮克传统，**善不一定胜，英雄早死，权力像毒品**。HBO 改编 Game of Thrones 现象级。
- **《冰与火之歌》(1996-)**：维斯特洛 7 王国 + 长城外异鬼 + 龙妈丹妮莉丝。主角 Ned Stark 第一季被斩首
- **范式贡献**：grimdark 范式 / "杀死主角" / 政治奇幻 / 影响了 The Witcher

### 9. Patrick Rothfuss (1973-, 美国) — Kingkiller 史诗
**核心理念**：用**学院奇幻 + 音乐 + 故事嵌套**，魔法分命名 / 召唤 / 塑形三律。
- **《风之名》(2007)**：Kvothe 流浪艺人回忆自己从音乐神童到大学的旅程
- **《智者的恐惧》(2011)** + 第三卷**未完结 (2017 至今)**
- **范式贡献**：学院奇幻 / 音乐 + 叙事 / Kickstarter 出版模式

### 10. Brandon Sanderson (1975-, 美国) — 魔法体系工程师
**核心理念**：Cosmere 宇宙 36+ 部，**每个世界有物理化学级别的魔法系统 + 三定律**。
- **《迷雾之子》(2006-)**：吞下金属 + 燃烧 = X 秒超能力。3 个魔法系统跨书可联
- **《飓光志》(2010-)**：10 部 Roshar 史诗宇宙
- **范式贡献**：硬魔法系统 / Sanderson's Laws of Magic / "魔法设计"方法论

### 11. J.K. Rowling (1965-, 英国) — 现象级奇幻商业化
**核心理念**：Harry Potter 7 部把奇幻从 nicher 变成**全民文化**。Hogwarts = 工业级别的世界构建。
- **《哈利·波特》7 部(1997-2007)**：11 岁孤儿 Harry 进 Hogwarts 学魔法。伏地魔 = 永生诱惑
- **范式贡献**：奇幻商业化 / 学校奇幻子类型 / 电影 + 主题公园 IP 工业

### 12. Susanna Clarke (1959-, 英国) — 维多利亚奇幻
**核心理念**：把奇幻写成**维多利亚小说风格**。
- **《英伦魔法师》(2004)**：1806 年英国唯一实用魔法师 Norrell + 学徒 Strange。用奥斯汀式散文 + 19 世纪注脚
- **《皮拉内西》(2020)**：一个人困在无尽迷宫的房子里
- **范式贡献**：历史奇幻 / 维多利亚散文风格 / BBC 改编剧

## 🏯 当代奇幻 (2010-)

### 13. N.K. Jemisin (1972-, 美国) — 破碎之地球
**核心理念**：连续 3 年获雨果奖最佳小说(2016-2018)，**种族 + 气候 + 后殖民**议题深度。
- **《破碎之地球》(2015-2017)**：Stillness 时代，地壳频繁爆发生存危机。Orogenes 种族 = 被奴役的地震控制者
- **《城市望不见的城市》(2020)**：纽约城变成活物
- **范式贡献**：雨果奖三连冠 / 后殖民奇幻 / 气候奇幻

### 14. 江南 (1977-, 中国) — 中国奇幻工业代表
**核心理念**：《龙族》IP 工业级商业化，**学院 + 都市 + 奇幻**路径。
- **《龙族》(2009-)**：路明非被揭示为龙族后裔，卡塞尔学院 = 屠龙者学校
- **《九州缥缈录》(2005-)**：尝试世界构建
- **范式贡献**：中国奇幻学院化 / IP 工业化 / 影响了《诡秘之主》

### 15. 阿来 (1959-, 中国) — 藏族魔幻现实主义
**核心理念**：《尘埃落定》傻子视角，**嘉绒藏区土司制度崩溃**，茅盾文学奖。
- **《尘埃落定》(1998)**：麦其土司傻瓜儿子第一人称视角，土司制度末期 1950 年代。傻子看穿权力但无力阻止
- **《云中记》(2019)**：汶川地震后祭师回归故乡
- **范式贡献**：少数民族奇幻 / 魔幻现实主义中国化 / 影响了 万玛才旦

### 16. 马伯庸 (1980-, 中国) — 历史解谜奇幻
**核心理念**：**把史料 + 推理 + 奇幻**熔于一炉。
- **《长安十二时辰》(2017)**：天宝三载上元节，长安 12 小时内挫败突厥狼卫。1:1 长安坊市结构做密室
- **《风起陇西》(2017)**：三国间谍战
- **范式贡献**：中国历史奇幻 / "史料 + 奇幻"范式 / 影响了《唐朝诡事录》

### 17. China Miéville (1972-, 英国) — 新怪谭旗手
**核心理念**：把 Lovecraft + 马克思主义 + 反殖民熔于一炉。**Bas-Lag 宇宙**。
- **《帕迪多街车站》(2000)**：新克罗布松工业城市，种族 + 魔法 + 科学混合
- **《使馆楼》(2009)**：两座城市占据同一空间，警察跨边界调查
- **范式贡献**：New Weird 范式奠基 / 影响了 Ann Leckie
