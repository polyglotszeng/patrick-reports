L4 2026 Design Trends — 5 directions

1. Bento Grid 布局

不规则大小方块, 8-12 块, 每块一个 feature. CSS Grid grid-template-areas. 参考: bentogrids.com

2. 滚动驱动动画

position: sticky + IntersectionObserver + Framer Motion. 不要全屏 scroll-jacking. 参考: Codrops / awwwards SOTD

3. Glassmorphism 进化

毛玻璃 + grainy gradient 背景, 不要纯透明, 加 noise texture. 参考: Recent (Godly)

4. 3D / WebGL 卡片

Three.js 装在 hero 一处, 全文其他克制 (避免性能灾难). 参考: Aceternity / splinetool

5. Neo-brutalism

厚黑边 + 鲜亮色 + 系统字体 + 反对"极简"过度审美. 参考: neobrutalism.dev / Stripe Press

⚠️ Pitfalls (8 documented)

1. Sandbox blocks `.dev` and `.design` TLDs — neobrutalism.dev, refero.design, interfaceindex.com. Don't mark as "down" — TLD-blocked.
2. CF wall detection = title-presence, NOT cf-ray header — real pages behind CF have cf-ray too. Only mark CF-WALL when body contains challenge marker AND no real title.
3. Vercel security checkpoint — SiteInspire, vercel.com/design. Same as CF but with x-vercel-id header.
4. Godly → Recent rebrand (2025) — godly.website rebranded; old links still 302. Current name is Recent.
5. Tailwind UI requires auth — 200 + JS shell, no real content. PASS-SHELL, not PASS.
6. curl -w "%{http_code}|%{size_download}|%{header_json}" unsafe to split on | — header_json has pipes/semicolons. Use curl -sLi + parse last HTTP status line.
7. Don't trust your own audit tool on first run — first probe reported 32/32 ❌ FAIL (curl parse bug), ground truth was PASS 5 min earlier. Smoke test on known-good site first.
8. PASS-SHELL ≠ FAIL — modern sites (Tailwind UI, Magic UI, Framer) are JS-rendered. Use browser_navigate for visual verify.
