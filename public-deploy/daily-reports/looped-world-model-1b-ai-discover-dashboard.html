④ ⭐ Looped Transformer = Reuse Same Block

核心机制

"Instead of stacking many unique layers, we could now repeatedly reuse the same block, the same transformer block, and we just loop it."

为什么

"Why should we suddenly not have here this deep deep transformer with multiple layers? Why only a transformer block to loop? Well, it is all about time, money, and the parameter we can reuse. And if you have one block where we loop over, this is a significant amount in the reduction of time and money and compute."

⭐ 新 scaling dimension

"It's not about bigger models. It's not about more data for the training or for the post-training, but it is a more iterative thinking that inside is happening the same model."

⑥ ⭐ Update Rule = 5 步 Recurrence

简化版 Update Rule

h_{t+1} = α · h_t + Action Injection + Non-linear Transformer Refinement
然后 repeat = T=100 times

5 步详解

  1. Prev hidden state = "because the system will not evolve here and have a discontinuity, nature will stay more or less the same"
  2. Inject observation or action = "if I picked up a bottle of Coke from the table"
  3. Non-linear transformer refinement = "this is where the intelligence comes into play, this is the third element where we have now everything from the transformer and MLP and non-linearity"
  4. Repeat = "how we loop, it couldn't be easier"
  5. T=100 loops = "all the shared parameters that are simply looped here for particular T T equal 100 times"

Patrick 启示

= "**Loop = 5 步 recurrence**" = "**L8 Principal 迭代哲学物理化**"

⑦ ⭐ Spectral Stability = Eigenvalues in [0,1] + KL

问题

"A latent update can easily explode or drift away."

修法 1: Spectral Stability

"They added a particular term, a spectral stability constraint on the retention part of the dynamics. And it parameterizes the I diagonal continuous time matrix with a negative entity entry, and this brings it back here the eigen values where in the interval between 0 and 1."
"The point is each loop step preserves the boundness instead of apply amplifying the state uncontrollable. So, everything is here in control."

修法 2: Kullback-Leibler Divergence

"We have a Kullback-Leibler divergence, and this makes sure we stay and we just go a little baby steps."

Patrick 启示

= "**Spectral stability + KL = 数值稳定**" = "**OpenSquilla MetaSkill 持续 refine 但不爆**"

⑧ ⭐ Variable Loop Count = Poisson Distribution

核心 insight

"During the training, the loop count is not fixed, and this is also something beautiful, you know? Normally, you have 100, 200, 500, but now this is really dependent, do we still need to continue to loop, or maybe we already have converged here to the best possible solution."

⭐ Poisson Distribution

"The pre-print samples the number of loop iterations from a Poisson distribution, so the model sees variable computational depths in its training phase, and hopefully then in the inference run, it will also apply this and says, hey, this is a simple run for 10 runs. I don't need to run this 500 times."

3 大优势

  • Robust for changing loop counts at inference
  • "Simple run = 10 loops" · "Complex run = 500 loops"
  • Total world model loss = observation + reward + continuation prediction

Patrick 启示

= "**Poisson loop count = 动态 depth**" = "**L8 Principal / OpenClaw dynamic = 自适应**"
