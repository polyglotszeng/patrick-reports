📂 Section-by-Section Summary

Part 1: 开源 motion transfer SOTA — Scale 2 (0:48 - 3:00)

  • 核心: 用 reference video 提取动作,迁移到任意目标视频(角色/动物/非人形态都行)
  • 亮点: 多角色动作可同时迁移;支持长 video;非人形态
  • 对比: 比 Live Portrait / DWPose 等老一代强
  • Patrick 视角: Game / VFX / 短视频创作有用,需自部署 cost-vs-quality 评估

Part 2: ZAI GLM 5.2 (3:01 - 9:00)

  • 定位: "quietly drops" = 不高调,但作者认为是当周最强开源 LLM 之一
  • 作者原话: "my favorite open source lab" — ZAI 一直是中文 LLM 开源主力 (GLM-4 / ChatGLM 系列)
  • 技术栈推测: 可能用 MoE + sparse attention (跟 Minimax M2 同趋势)
  • 背景: ZAI 跟清华 Zhipu AI 关系密切,中国 LLM 开源生态 top tier
  • Patrick 视角: 中文 LLM 选型,GLM 系列是默认候选之一

Part 3: Kimi (Moonshot) 新模型 (3:15 - 4:30 + 18:11 - 19:30)

  • 3:15 段: "absolute beast" — 新模型,具体规格未细说

  • 配套生态: "Kimi Code" (18:11 提及) — 类 Claude Code 的 coding agent
  • Patrick 视角: Kimi K2 是 Anthropic / OpenAI 之外的中文 coding agent 备选

Part 4: Google 两连发 (9:04 - 18:00)

  • 9:04 起: "incredibly fast open source model" — 推测是 Gemma 3 / Gemini Flash 变体
  • Realtime translator: 70+ 语言,免费,实时
  • 训练机制: 9:40 提及"predicting the next word... whole block" — autoregressive 或 block-wise denoising
  • Patrick 视角: Google 开源策略 2024-2025 加码,Gemma 3 27B 已是 Mac 本地强候选

Part 5: Minimax M2 开源 (22:20 - 24:00)

  • 本地可跑: 作者称可在消费级硬件跑

  • Patrick 视角: M2 / M3 (MiniMax-M3) 系列已自部署,稀疏注意力是 token 经济性关键

Part 6: 4D Full-Body Avatars (预告段 + 24:00)
