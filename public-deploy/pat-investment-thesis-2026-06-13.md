# PAT 个人 AI Agent 经济学投资分析

**Patrick 投资框架**: NQ-100 Plan E 60% 5 因子化 + Regime Filter (43.2pp 改善 vs QQQ) + 25% 现金 + 15% 机会性. AI Agent 经济学方向 = 4 玩家洗牌, 个人 **不直接入股** (公司都未上市或 PE 锁死), **走 ETF + 行业指数** 路线.

**本分析** = 把 AI Agent 经济学 4 玩家判断 + 12-18 月窗口预测, 转化为 **可执行的 2-3 个 ETF + 4-5 行业个股** 配置.

## 1. 4 玩家 12-18 月窗口预测 (核心判断)

### 1.1 Anthropic (Routines + Managed Agents + 视觉 UI 待发)
- **当前**: 未上市, 估值 $40B+ (2025-09 报道), Claude Opus-4.7 + Sonnet 4.5 卖 token
- **12-18 月窗口**: 推出 visual drag-drop UI (Nick Saraev 视频 3 段 213-217 预测) → **直接 Kill n8n**
- **关键事件**: 2026 H2 visual UI 发布 (高概率, Nick 推断 Anthropic "probably already working on this")
- **影响**: 4 玩家里 **唯一全栈** (model + agent + UI), 估值会冲到 $80-150B
- **投资路径**: 无法直接投资. **间接**: Microsoft (持有 49% OpenAI, 间接参股 Anthropic 边缘), Google (TPU 供应 Anthropic)

### 1.2 n8n (visual drag-drop 主流)
- **当前**: 估值 $1B+ (2024-12 $200M 融资), ARR ~$50M (估, 公开数据少)
- **12-18 月窗口**: 被 Anthropic visual UI 挤压, 6-12 月客户流失 (保守)
- **关键风险**: 2024 估值 $1B 是 high watermark, 2026 H2 估值大概率砍 30-50%
- **投资路径**: 未上市. 公开市场无 direct exposure. **间接**: 1 个软银/红杉持股 ETF (不可得)

### 1.3 Lindy (托管 AI agent)
- **当前**: $49-299/月, ARR 未公开, 私人公司 (Crustdata 显示 < 50 员工)
- **12-18 月窗口**: 跟 Routines 直接竞争, 价格会被压 (Routines $0-50/月 vs Lindy $49-299/月)
- **关键风险**: 客户可能迁到 Anthropic Routines (便宜 5-10x, 1 行 prompt 替代)
- **投资路径**: 未上市, 无公开市场 exposure

### 1.4 Artisan AI (卖实施服务)
- **当前**: $5M ARR (2025-04 TechCrunch, 已纠错), 250 客户, 57 人, $36.5M 融资
- **12-18 月窗口**: 12-18 月内 visual UI 出 = 客户自服务, 服务商模式不可持续
- **关键风险**: Nick 自己在视频 3 段 215-217 提到 "either service providers like ourselves... will basically be able to use this managed infrastructure" = **服务商要转型** (卖 Managed Agents 实施而非 n8n 实施)
- **投资路径**: 未上市, 无公开市场 exposure

### 1.5 其他玩家 (5 Falsification 跨检)
- **OpenAI**: 仍是头部 model, 但 agent 层动作慢. Operator + 5 月 Codex updates. **风险**: Anthropic Managed Agents 出 = OpenAI 压力
- **Microsoft Copilot Studio**: enterprise 大客户, 跟 n8n/Power Automate 直接竞争
- **Google Vertex AI Agent Builder**: 跟 Lindy 路线类似, 2B 客户群
- **中国玩家**: 字节扣子, 阿里通义, 百度千帆, 腾讯元器. **中国网络限制** = 不能用 Claude, 但 Anthropic 仍占 US 市场

## 3. 12-18 月 2 场景预测 (Plan A vs Plan B)

### Plan A: Anthropic visual UI 6-12 月出 (高概率 60%)
**情景**: Nick 推断正确, 2026 H2 看到 drag-drop UI demo, 2027 H1 全面上线
**影响**:
- n8n 客户 30% 流失 (6 月), ARR 砍半, 估值 $1B → $400-500M
- Lindy 客户 20% 流失 (便宜 5-10x), ARR 持平或微降
- Artisan 客户自服务, ARR 持平 (靠 add-on 服务), 估值持平
- **赢家**: Anthropic (估值翻倍), Microsoft (持有 OpenAI 49%), Google (TPU + Gemini 间接)

**Patrick 仓位调整** (Plan A 触发):
- **加仓 GOOGL +3%** (重复权重到 13%) = Anthropic visual UI 出 → Google 同步发力
- **减仓 PLTR 1%** (从 3% → 2%) = Anthropic 自家数据集成更强
- **维持 SMH/CRWD** = 不变

### Plan B: Anthropic visual UI 12-18 月未出 (中概率 30%)
**情景**: 内部优先级低, 或视觉 UI 技术难度大, 或 12-18 月窗口内没出
**影响**:
- n8n 估值持平, Lindy 估值上行 (Routines 还没视觉化, Lindy 仍有优势)
- Artisan 客户流微跌 (没新冲击, 但也没加速)
- 4 玩家继续僵持, 估值横盘

**Patrick 仓位调整** (Plan B 触发):
- **加仓 n8n/Lindy 间接 (CRWD +2%, PLTR +2%)** = Lindy 估值上行 → PLTR/CRWD 跟着 AI agent 安全/集成需求上涨
- **减仓 SMH 2%** (从 5% → 3%) = 算力需求增速放缓

### Plan C: 黑天鹅 (10% 概率)
**情景**: Anthropic API 出大事 (DDoS/合规危机/核心团队离职), 估值砍半
**影响**:
- 4 玩家全盘受影响, AI agent 板块 1 月 -30%
- **Patrick 仓位保护**: 25% 现金 + Plan E 60% 分散, 5-10% 损失上限
- **加仓机会**: 现金 25% → 10% 抄底, 12 月后回弹 30-50%
