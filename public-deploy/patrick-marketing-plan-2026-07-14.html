§0核心论断 (1 段话)

结论Patrick 不缺策展产出，缺"策展人身份"的可见性 + 信任锚点。

264 reports + 985KB launch-atlas + multi-perspective salon + 19 skills = 已经有海量 substance，但没人知道 Patrick 是谁、为什么读 Patrick 跟读别人不一样。

→ Marketing 第一性原理：卖的不是内容，是 "Patrick 担保 + Patrick 立场 + Patrick 研究深度"。

→ 反 audit §6 里的"商业化 #2" 提案：不卖 paywall，卖信任凭证 + 标注层 + 慢工订阅。

2.1 A 路径：JSON API B2B 详解

做什么

  • 把现有 reports.json 转成机器友好的 JSON API（每条 entry 加 provenance / curator / cross-ref 字段）
  • 免费：人类用户读 HTML / 跟订阅
  • 收费：api.patrick-reports.dev/v1/curated license — 调用频率 > 1000/月 / 商业用途 / badge 嵌入授权

定价

  • Free tier: 1000 calls/mo + 个人非商用
  • Indie: $29/mo - 10K calls + 商用许可
  • Team: $99/mo - 100K calls + SLA + 自定义 badge
  • Enterprise: $500+/mo - 无限 + co-marketing

合规立场

API 是工具销售（类比 Cloudflare / Stripe），不是内容付费墙。公网 HTML 永远免费 + 永远完整。机器访问的人用机器价钱，人的阅读权不被买卖。

2.2 B 路径：1-on-1 策展咨询详解

做什么

  • 目标客户：已经有 newsletter / blog / 公众号 但"内容好但没节奏 / 没立场 / 没深度"的人
  • 服务包：4 次 1h 视频 + 1 份策展 audit 报告（基于 audit §7 的 10 prompts） + 30 天微信答疑
  • 可加价：陪跑 3 个月（$8K-15K）

合规立场

咨询 = 卖 Patrick 时间，不卖 Patrick 内容。Patrick 的 264 reports / 5 frameworks 永远公开。这是"教练 + 顾问"模式（类比 Stratechery Ben Thompson 的 sponsorship talks），不是"独家内容"。

2.3 C 路径：Patrick's Quarterly 详解

做什么

  • 季度精选合集，从 264 reports 里挑 30 条 + 重写注解 + 加 Patrick 当年思考
  • 4 个 deliverable：HTML（公网免费）+ 印刷 PDF + EPUB + iMessage 摘要
  • 第一期做 2026 Q2（4-6 月）= 已经覆盖 60+ reports
  • 不卖订阅：公开免费 + 印刷版预约制 + 限量 100 本实物（礼物感）

合规立场

HTML 永远免费 + 永远公开。印刷版是"实体礼物"（类比 Marginalian 的 Brain Pickings 早期年度合订本），不是 paywall。限量 = 稀缺感 ≠ 准入门槛。
