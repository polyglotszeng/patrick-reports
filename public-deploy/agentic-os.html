06FAQ

The questions people actually ask.

Is this really free? What's the catch?

Yes, really free. The code is MIT licensed — copy, modify, sell, ship. No attribution required. The only "cost" is your own model API key (Claude ~$5/mo for casual use, or free with local LLMs). Julian's $67/mo isn't the code — it's the community + convenience. If convenience matters more than cost, pay Julian. If you want to learn and own, use this.

Why would I use this over Julian's?

You use Julian's if: you want zero setup, don't mind paying monthly, and value his community. You use this if: you want to understand the system, customize it, or build your own product on top. Both are valid. The Hermes core is the same.

Is this the same as Hermes Agent?

Yes — Patrick's Agent OS is built ON TOP of Hermes (the open-source AI agent). It's a curated, pre-configured setup. Hermes does the AI work; Patrick's pack adds the cron jobs, memory system, PITFALLS, and skills library on top.

Can I sell my own version?

Yes. MIT license. Build a niche Agent OS (for SEOs, e-commerce, lawyers, doctors, etc.) and charge whatever you want. Patrick's pack is your starting point.

Does it run on Windows / Linux?

Patrick's pack is macOS-first (uses launchd). The Hermes core is cross-platform. A Linux version with systemd timers is on the roadmap. Windows is untested.

How is this different from just using Claude Code or Cursor directly?

Claude Code / Cursor are interactive coding tools. This is a full operating system for AI agents — it runs tasks in the background, schedules them, learns from mistakes (PITFALLS), and remembers context across sessions. Use both: this for automation, Claude Code for interactive coding.
