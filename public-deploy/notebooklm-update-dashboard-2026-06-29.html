  5. 仅 Google AI Ultra 订阅者 + Workspace Ultra 用户先得, 输出仍需人审.

📦 升级 3: 零源起始 (zero source mode)

[05:42] 旧模式: 必须先喂源. 新模式: 打开空 notebook, 输入问题 → 系统上网找源 → 列候选供 approve/skip → 基于批准源建完整 notebook.

🔁 流程示意

用户输入 question (空 notebook)
         ↓
NotebookLM 上网检索 (相关 web sources)
         ↓
展示候选源列表 (含 URL + 摘要)
         ↓
用户 approve / skip (per-source)
         ↓
基于已批准源建立完整 notebook
         ↓
后续 RAG / 产出仍走标准 NotebookLM 流程
概念类比: 概念 ≈ Perplexity (联网 RAG) + Notion AI (产出文档) + 信任 checkpoint (approve/skip). 与 [[distributed-research-playbook]] + [[graphify]] 思路一致.
⚠️ 局限提示: zero source 模式下, topic + goal 必须精确 ([08:49] 主播承认). 模糊 prompt = 找错源.
