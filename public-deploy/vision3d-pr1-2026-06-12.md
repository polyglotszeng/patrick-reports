# PR #1 — Split LightPollutionOverlay.swift into Models/ + Views/

> **PR status (2026-06-12):** Files written directly to project.
> Project is NOT a git repo (`fatal: not a git repository` at
> `/Users/patrick/Projects/Vision3D`). The "PR" is a manual patch
> + ADR; not a real GitHub/GitLab PR.

## Summary

`Views/LightPollutionOverlay.swift` was 17 KB / 426 lines containing
1 enum, 1 struct, and 4 View structs. Per `CONTEXT.md` boundary
("Models/ MUST be Codable structs, no behavior"), this was an
architecture violation.

This refactor splits the file into 3:

| File | Lines | Type | Notes |
|------|-------|------|-------|
| `Models/BortleScale.swift` (new) | 70 | Data | Enum with Bortle classes 1-9 |
| `Models/ObservationSite.swift` (new) | 110 | Data | Now `Codable` (was missing) |
| `Views/LightPollutionOverlay.swift` (modified) | 250 | View | 4 View structs only |

## Motivation

Without this split:
1. `CONTEXT.md` says "Models/ MUST be Codable". `ObservationSite` was
   not Codable, so it couldn't be persisted in `SharedUserData`
   (ADR-0002). This was a latent bug.
2. Contributors couldn't tell at a glance whether `BortleScale` was
   data or UI — both lived in a Views/ file.
3. The file was large enough that scrolling was required to find
   anything.

## Changes (real, applied)

### Added: `Models/BortleScale.swift`
- Extracted `enum BortleScale` (lines 1-70 of old file)
- Added file-header comment linking to ADR-0005

### Added: `Models/ObservationSite.swift`
- Extracted `struct ObservationSite` (lines 72-157 of old file)
- **Added `Codable` conformance** with custom `CodingKeys` + init
  (preserves `id` across decode — was a latent bug)

  the `let id = UUID()` default

### Modified: `Views/LightPollutionOverlay.swift`
- Removed lines 1-157 (data moved out)
- Kept 4 View structs (158+)
- Added file-header comment pointing to ADR-0005

### Added: `docs/adr/0005-lightpollutionoverlay-split.md`
- The full decision record (Context, Decision, Consequences,
  Alternatives, When to revisit, Migration notes)
