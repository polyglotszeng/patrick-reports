## 🕯️ 黄金奠基 (1841-1920)

### 1. Edgar Allan Poe (1809-1849, 美国) — 侦探小说之父
**核心理念**：1841 年《莫格街谋杀案》创造 Dupin，**推理范式的发明者**。
- **《莫格街谋杀案》(1841)**：世界上第一篇侦探小说
- **《失窃的信》(1844)**：最聪明藏匿 = 把信放在最显眼的地方
- **范式贡献**：侦探小说 + 推理方法 + 心理悬疑 + 黑色恐怖 = 现代通俗文学"创世者"

### 2. Sir Arthur Conan Doyle (1859-1930, 英国) — 福尔摩斯
**核心理念**：把 Poe 范式扩展为 60 篇 + 4 部小说，**"演绎法"** 成为推理代名词。
- **《血字的研究》(1887)**：福尔摩斯 + 华生初登场
- **《巴斯克维尔的猎犬》(1902)**：Devon 荒原诅咒猎犬
- **范式贡献**：侦探 + 助手范式 / 演绎法 / 烟雾弹 + 误导

### 3. Agatha Christie (1890-1976, 英国) — 本格女王
**核心理念**：66 部推理 + 14 部短篇集，**"侦探小说黄金时代"中心**。
- **《无人生还》(1939)**：10 人按童谣顺序死在士兵岛，凶手也是受害者
- **《东方快车谋杀案》(1934)**：12 个乘客每人一刀，波洛选"陪审团"结局
- **《罗杰疑案》(1926)**：叙述性诡计开山 — 凶手是叙述者
- **范式贡献**：本格范式 + 公平性原则 / 卧底式叙述诡计 / 销量 20 亿仅次于莎士比亚

### 4. G.K. Chesterton (1874-1936, 英国) — 布朗神父
**核心理念**：反 Sherlock 的"演绎法"，**用心理学 + 神学**理解罪犯。
- **《布朗神父探案集》(1910-1935, 51 篇)**：矮小神父用"忏悔者心理学"破案 — 他自己也有犯罪冲动
- **范式贡献**：心理侦探 / 道德悬疑 / 反 Sherlock 的"凡人之智" / 影响了 Umberto Eco

## 🚬 硬汉 / 冷硬 (1920-1970)

### 5. Dashiell Hammett (1894-1961, 美国) — 黑色小说之父
**核心理念**：Sam Spade + Continental Op，**把侦探小说从"解谜游戏"变成"社会黑暗面"**。
- **《马耳他之鹰》(1930)**：传说中雕像实际是假的。Humphrey Bogart 1956 影史经典
- **范式贡献**：硬汉派范式 / 黑色电影 / "私家侦探"子类型

### 6. Raymond Chandler (1888-1959, 美国/英国) — 硬汉派诗人
**核心理念**：Philip Marlowe 7 部，**把硬汉派变成文学**。
- **《漫长的告别》(1953)**：友谊 + 失落的加州 + 写作的孤独。村上春树译本让他在日本封神
- **《再见，吾爱》(1940)** + 《大眠》《高窗》等 7 部
- **范式贡献**：硬汉派文学化 / "马洛"成为侦探代名词

### 7. Mickey Spillane (1918-2006, 美国) — Mike Hammer 暴力
**核心理念**：比钱德勒更暴力、更直接，**冷战时期美国男性焦虑的化身**。
- **《审判者》(I, the Jury, 1947)**：朋友 Jack 被杀，Mike Hammer 追查凶手包括开膛手 + 黑社会 + 变性人
- **范式贡献**：暴力硬汉范式 / 反共冷战小说 / 影响了 Lee Child

## 🌏 全球化 (1970-至今)

### 8. 东野圭吾 (1958-, 日本) — 日本社会派 + 推理产业化
**核心理念**：90+ 部，**把"动机"作为推理核心**。
- **《嫌疑人 X 的献身》(2005)**：数学天才石神帮邻居处理杀夫案，结局揭示**爱 = 自我毁灭**
- **《白夜行》(1999)**：两个小孩 1973 年犯下命案，余生 19 年互相利用
- **《恶意》(1996)**：有些犯罪无动机，只是纯粹的恨
- **范式贡献**：日本社会派 + 动机推理 / "献身"概念 / 中国销量 1000 万+

### 9. Ed McBain (1926-2005, 美国) — 警察程序派
**核心理念**：55 部 Isola 警察程序系列，**把警察写成一个机构**。
- **《87 区警察》(The 87th Precinct, 1956-2005, 55 部)**：多侦探协同办案
- **范式贡献**：警察程序范式 / 影响了 Law & Order 等警察剧

### 10. Stieg Larsson (1954-2004, 瑞典) — 千禧系列 / Nordic Noir 旗手
**核心理念**：《龙纹身的女孩》三部曲，**黑客女侦探 + 瑞典福利国家阴暗面**。
- **《龙纹身的女孩》(2005)**：财经记者 + 19 岁黑客 Lisbeth Salander 调查 40 年前失踪案
- **范式贡献**：Nordic Noir 范式 / 黑客女侦探 / 销量 1 亿+

### 11. Jo Nesbø (1960-, 挪威) — Harry Hole
**核心理念**：12 部，**奥斯陆 + 雪 + 心理黑暗**。21 世纪北欧 Noir 代表。
- **《雪人》(2007)**：初雪降临奥斯陆，25 年前开始的连环。Harry Hole 嗜酒 + PTSD
- **范式贡献**：北欧黑色 + 心理变态

### 12. Michael Connelly (1956-, 美国) — Harry Bosch 警察程序
**核心理念**：35+ 部，**洛杉矶警局 LAPD 罗伊中尉**。前《洛杉矶时报》犯罪记者。
- **《黑色回声》(1992)**：LAPD 警探 Harry Bosch 调查隧道命案。Amazon Bosch 剧 7 季
- **范式贡献**：程序正义范式 / LAPD 详细描写 / 新闻 + 小说 双专业

### 13. Tana French (1973-, 爱尔兰) — 都柏林凶案组
**核心理念**：8 部，**每部换主角警探 + 心理深度**。文学化推理 + 心理学融合。
- **《林中秘径》(In the Woods, 2007)**：童年创伤 + 调查者本身是嫌疑人
- **范式贡献**：都柏林学派 / 心理悬疑 + 推理 / 影响了 Gillian Flynn

### 14. 紫金陈 (1982-, 中国) — 中国当代推理产业旗手
**核心理念**：《坏小孩》《长夜难明》《低智商犯罪》系列，**中国城乡转型 + 互联网 + 房价**。
- **《坏小孩》(2014)**：三个孩子策划杀害施压大人。改编剧《隐秘的角落》(2020) 现象级
- **《长夜难明》(2017)**：检察官江阳为冤案追查 10 年。改编剧《沉默的真相》
- **范式贡献**：中国当代推理 / 城乡转型议题

### 15. James Ellroy (1948-, 美国) — LA Quartet
**核心理念**：《黑色大丽花》《洛城机密》《白色爵士》《美国秘史》，**洛杉矶 1940-60 年代黑色史诗**。
- **《黑色大丽花》(1987)**：1947 年黑色大丽花 Elizabeth Short 肢解案。现实 + 阴谋论
- **范式贡献**：超现实黑色 / 影响了 L.A. Confidential 电影 / "阴谋论 + 警察程序"范式
