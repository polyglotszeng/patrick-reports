💡 创作方法
García Márquez "孤独的拉丁美洲" = 百年家族编年史范式；Borges "迷宫 + 镜 + 图书馆" = 概念小说；Bolaño 长篇叙事 = 实验小说；Cortázar 可玩小说 = 交互叙事
🛠️ 知识工程
Borges 巴别图书馆 = 知识穷尽悖论；Carlos Fuentes 时间三部曲 = 跨时空叙事；Allende 女性叙事 = 多重视角
🌏 跨文化视角
魔幻现实主义 = 西方对非西方时间观的接受；Rulfo 墨西哥荒原 = 拉美现代性反思；Bolaño 跨国流亡 = 全球化作家
🔁 Loop 启示
拉美作家用 "一个家族 + 一片土地 + 一百年的时间" 起手 → Patrick 的研究单元也应是 "一个主题 + 多个时代 + 多个视角" 的长篇结构
