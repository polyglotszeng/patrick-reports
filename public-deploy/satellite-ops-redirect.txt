Satellite Ops - Global Satellite Realtime Dashboard

Live URL: https://brewpage.app/public/azVgnRbHBi

This is a Cloudflare Pages Worker-shadowing workaround.

The page is currently hosted on BrewPage (15-day TTL).
Source: polyglotszeng/patrick-reports repo, commit b5ff3acc
File size: 39062B (single-file HTML, no external deps)

Architecture:
- index.html: vanilla JS + Leaflet 1.9.4 (CDN) + OpenStreetMap tiles
- Polling: FIRMS NASA (5 min cache), CelesTrak TLE (1h), AISStream.io (live)
- AOI watchlist: 18 regions (Taiwan, Red Sea, Ukraine, Persian Gulf, etc.)
- Anti-cache: no-store headers + 4 fresh sources per layer

Status: in production hardening loop (Loop 7/8)
