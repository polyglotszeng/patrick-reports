The system

A shared brain, so the swarm gets smarter together

Every agent reads and writes the same context, backed by a layered memory system and model routing — an Agent Operating System that keeps evolving instead of resetting.

Memory

Six-layer memory

L1 logs → L2 weekly → L3 long-term → L4 dormant → L5 roles → L6 tools. Agents accumulate and recall, instead of starting cold.

Graph

SAGE memory graph

Semantic clustering and reinforcement-managed edges organize scattered findings into a graph the swarm can reason over.

Routing

Multi-model routing

Scatter, ensemble, and pipe modes dispatch each task to the model that fits it best — not one model for everything.
