6 个 Milestone 流程

每个 milestone = 1 prompt + git commit + 截图自检

① 清理场景 0:55

删 accessories，保留 third-person character + 灯光天空。Claude 会问合理问题（"keep lighting in sky?" → "Yes"）

② Endless runner 核心 6:00

清空关卡，生成 infinite pass。Claude 主动用 Unreal Claude 截 viewport 验证 — "plane + character standing on it"。变量：tile length, path length。

③ 自动跑动 + 相机 8:00

改自动前进（不按 W），相机从顶部俯视。Python 脚本调相机位置。

④ 三车道 + A/D 切换 9:15

逻辑 3 lane 切换。Blueprint spaghetti 重灾区 — 节点位置永远不完美。

⑤ 障碍 + 金币 + 随机化 10:45

单任务耗时 15 min / 14,000 tokens（Opus 4.8） — 需选 milestone 拆分。

⑥ UI（HUD）12:00

分数、金币、Game Over、Retry 屏。设计"doubtful" — 概念先跑通，美观后补。

# Bug 现象 修复方式
1 走左右边 = game over（误判） 截图 → Claude 调整 collision trigger 范围
2 桥之间有 gap（穿模） 截图 → Claude 调 tile length + 拼接逻辑
3 障碍钻入 bridge 截图 → 修 + 顺便关 toon shader（多余效果）

💡 关键 insight

Claude 自带 viewport screenshot tool，可独立 "is character on plane?" 自检。Stefan 把这做成 milestone 完成的硬性条件：截图显示正确 → 才 commit。

最终评估：Stefan 诚实结论

✅ 优点

  • 逻辑质量好：3 lane 切换、障碍、金币都跑通
  • 截图自检循环让 Claude 自我 debug
  • 可调试：教新手 "Ask why? What is this node?"
  • 对初学者加速极大

❌ 限制

  • Blueprint 节点位置永远不完美（always spaghetti）
  • 泛 prompt 必失败 — "make a game" 不可扩展
  • 14K tokens / 15 min = 贵 + 慢，要选 milestone
  • 不擅长图像生成，资产/背景用 ChatGPT image / Meshy

适合谁 / 不适合谁

✅ 适合❌ 不适合
懂逻辑 / 概念 / 知道 game 该怎么搭的人 完全没想法 + "make a game"
想把 UE 当工具加快原型的人 想要 pixel-perfect blueprint 美观
愿意截图 debug + 写详细 prompt 想要"一次 prompt 出完整游戏"
学习者（问 "why this node?"） 想要免费 Claude（Opus 14K token 烧钱）
"I definitely think that is gonna be a full replacement. If you can describe the logic that you want, it can be very, very helpful, especially for the beginners."

复用要点：我能直接用

1️⃣ CLAUDE.md 模板化

核心："use MCP X for screenshots, MCP Y for blueprints, commit after every milestone"。完整版 Stefan 在视频描述里给订阅者。

2️⃣ Milestone = commit 节奏

UE 大项目拆 6 个 milestone，每个 1 prompt + commit。失败可回滚。

3️⃣ 截图驱动 debug

Claude 自带 viewport screenshot tool，milestone 完成 = 截图证据。

4️⃣ 资产走外部

ChatGPT image / Meshy 预生成，不要 Claude 做资产生成。

5️⃣ 任务边界

只让 Claude 做逻辑 + 集成，不做资产生成 + 蓝图美化。

6️⃣ 替代 Claude

Cursor / Codex / OpenCode 走同 MCP 协议（Unreal Claude + Vibe UE 是 open source），不锁 Claude API。
