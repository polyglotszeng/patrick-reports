Loop vs Prompt 核心机制

❌ Prompt（旧）

你 = 引擎

  • 持续 babysit the thread
  • Hands-on 整段时间
  • One mistake → restart
  • Capped by attention / energy

✅ Loop（新）

你 = 架构师

  • 设一次（~1 min）
  • Run whilst sleep / work
  • Auto iterate
  • Capped by loop design

"One makes you the engine, the other makes you the architect."

⚔️ Loop vs Prompt 哲学

维度Prompt（旧）Loop（新）
谁主导 你 = 引擎 你 = 架构师
时间投入 持续 babysit 设一次（~1 min）
注意力 constant set and forget
产出上限 capped by attention / energy capped by loop design
失败处理 one mistake → restart thread loop iterates automatically
multitasking ❌ 不行 ✅ 跑 loop 时可做其他
结果 wake up to "still working" wake up to "done"

🛡️ Doer + Judge（2 角色架构）

"The catch here is what most people miss, which is a loop only works if something other than the agent decides when it's done."
"The agent that wrote the code can't be the one that grades it. It will always say, 'Looks good to me.'"

2 角色

角色任务关键
Doer Agent acts / builds / writes / searches 实际 work
Judge Separate model grades against goal "doesn't need to be smarter, just independent"

Loop 流程

Define what done looks like
        ↓
   Doer acts (build / write / search)
        ↓
   Judge checks against goal
        ↓
   Pass → Ship ✅
   Fail → Loop back to Doer 🔁
        ↓
   Repeat until goal met

核心优势

  • ✅ "Scales because one loop or 100 is the same effort from you"
  • ✅ "Repetitive work stops being your problem"
  • ✅ "Everything improves together"

结构

🧠 Memory Loop（Positive Feedback Loop）

机制

"Every time I speak to my agents, they automatically log outputs inside our Obsidian memory."

"These are all memories I've never built or touched any of them."

Positive Feedback Loop

Agents work → output → Obsidian memory
                              ↑                    ↓
                              ←←←←←←←←←←←←←←←←←←←←
                                  (next loop)

效果

  • ✅ Memory improves
  • ✅ Context improves
  • ✅ More notes
  • ✅ "Better system beautifully organized"

"My agents work in a positive feedback loop because when they are generating outputs, they plug it into this Obsidian galaxy. And then, when they're coming up with answers, they pull from that Obsidian galaxy."
